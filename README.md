# Employee Dashboard

This project displays employee information using JavaScript array methods. It was built as part of a Coursera hands-on lab.

## Features
- Show all employees
- Calculate total salary
- Filter by HR department
- Find employee by ID

## How to Run
Open `index.html` in a browser and use the buttons to interact with the dashboard. Console output will show the results.
